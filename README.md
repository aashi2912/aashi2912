## 🎯 How I Think About Product

> I don't believe in building features nobody asked for. The best products come from sitting with users long enough to understand what they're *actually* struggling with — not what they say they want in a meeting.
>
> For me, AI isn't the product. It's a tool. The product is the problem you're solving and how seamlessly you solve it. If the user has to think about the AI, you've already lost.
>
> Ship small, measure honestly, and have the guts to kill things that aren't working.
